How to Download Source files with Eclipse & git: We open Eclipse Java IDE:

Step1:Go to 'Window'
Step2:Go to 'Show View'
Step3:Go to 'Other...'

The Show View Window:Here we select an new window/tool

We open the folder with the name "Git"
Step1:Select the "Git Repositories"
Step2:Click the OK Button

The Git Repositories:Here we add our repositories.

Step1:We click into the "Clone a Git Repository"

The Clone Git Repository Window:From here we check out our branches

The URL for CorePack:https://github.com/L2jLastWarrior/CorePack
The URL for DataPack:https://github.com/L2jLastWarrior/DataPack
Step1:Select one of two link and pasted into URL textbox
Step2:Click the "Next>" Button

The Clone Git Repository Window(part2):From here we select the chronicle that we need to download.

Step1:We select our Chronicle that we need(we can select more than one)
Step2:Click the "Next>" Button

The Clone Git Repository Window(Final part):Now we already to download our sources into Local Workspace folder.

Step1:We check this options to create an new project
Step2:Click the "Finish" Button to start the download proccess.

How to Download Source files with Eclipse & svn: We open Eclipse Java IDE:

Step1:Go to 'Help'
Step2:Go to 'Eclipse Markplace...'

Eclipse Markplace Window:Here we search the SVN package

Step1:In find field write "svn"
Step2:Click in "GO" Button

Eclipse Markplace Window(part2):Our package name is "Subversive -SVN TEAM Provider"

Step1:Click in "Install" Button

Eclipse Markplace Window(part3):Package Installation

Step1:Click in "Change"Button
Step2:Click in "Change" command

Eclipse Markplace Window(part4):Package Installation - Select aditional Packages

Step1:Select all packages
Step2:Click in "Confirm>" Button

Eclipse Markplace Window(part5):Package Installation - Licenses

Step1:Select "I accept..."
Step2:Click in "Finish" Button

Eclipse Dialog - Software updatesHere we need to restart our IDE:

Step1:Click in "Yes" Button

Show pallet:SVN pallet

Step1:Go to "window"
Step2:Then go to Show View
Step3:Click into "Other..."

Show pallet(part2):SVN pallet

Step1:We open the "SVN" folder
Step2:Click in "SVN Repositories"
Step3:Clikc in "OK" Button

New Repository Location:Here we add our repositories

Step1:Click into SVN-Database icon
Step2:Click in "Data+" icon with name "New Repository Location"

New Repository Location:Here we add our repositories

The URL For CorePack:https://subversion.assembla.com/svn/corepack/trunk
The URL for DataPack:https://subversion.assembla.com/svn/datapack/trunk
Step1:Select one of two link and pasted into URL textbox
Step2:Select "Use a custom label
Step3:Give an custom name for project
Step4:Click in "Finish" Button

Repository Location Properties:

Step1:Click in "Yes" Button

Download Server FilesWe use check out to download

Step1:Right click into our Repo
Step2:Click in "Check Out"
When operation complete our project will be in your workspace
